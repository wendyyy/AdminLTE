How to select params to launch your build
  • "Node": choose a available node to run your test
  • "Branch": this is automation code branch
  • "Server": don't forget to select your test env (qacand or qapatchpreview)
  • "MailTo": write down your email address
  • "TestFor": for debug you can just write down "debug"
  • "StartOdataCheckBeforeTest": "yes" means it will start Odata API status check before launch the automation test
  • click "Build" to launch the test
